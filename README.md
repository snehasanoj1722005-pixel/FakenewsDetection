# Fake News Detection

## Description
This project detects fake news using machine learning techniques.

## Technologies Used
- Python
- TF-IDF Vectorizer
- Logistic Regression

## How it Works
1. Input news text
2. Text preprocessing
3. Convert text into numerical form
4. Model predicts Real or Fake

## How to Run
1. Run the Python file
2. Enter news text in terminal
3. Get prediction output
